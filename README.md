# Simple Webpage Project

This project is a basic HTML and CSS website created as part of an early web development assignment. It demonstrates how to structure a webpage, apply styling through an external stylesheet, and include images in a clean layout.

## Features
- Basic webpage structure using semantic HTML
- External CSS file for styling
- Image folder for organized asset management
- Simple, readable layout

## What I practiced
- Writing clean HTML
- Linking and organizing CSS files
- Using images in a webpage
- Structuring a project with separate folders for assets

## Technologies Used
- HTML5
- CSS3

## How to View
Open `index.html` in any web browser to view the webpage.
